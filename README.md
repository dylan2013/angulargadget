Angular Gadget
====

## 前置要求

* 安裝 Node.js
* 安裝 Yarn (建議安裝)
* 安裝 VSCode
* 安裝 Angular CLI。

## 安裝 Angular CLI (command line interface)
```sh
yarn global add @angular/cli
```

## Quick Start
* `yarn` #安裝套件
* `yarn start` ＃啟動 angular server。
* 連到 `http://localhost:4200` 開始 Debug。

## Deploy
* yarn build
* 把 dist 所有檔案部署出去。
