Angular Gadget
====

## 前置要求
* 安裝 Node.js
* 安裝 Yarn (建議安裝)
* 安裝 VSCode
* 安裝 Angular CLI (`yarn global add @angular/cli`)。

## 開始開發
* `git clone https://github.com/gildarts/angulargadget mygadget`
* `cd mygadget`
* `yarn` #安裝套件
* `yarn start` ＃啟動 angular server。
* 連到 `http://localhost:4200` 開始 Debug。

## Deploy
* yarn build
* 把 dist 所有檔案部署出去。
